# jobs-nu

 
 ## Run server & client
  _Run Server_
1. In server : Port 8080
    ```sh
     nodemon src/app.js
    ```
  _Run client_

2. In client : Port 8080
    ```sh
     npm run start
    ```    
## Install npm server & client // Blackend & Frontend
  ```sh
     npm install
   ```
## URL project
 ```sh
   http://localhost:8080/login
   ```
